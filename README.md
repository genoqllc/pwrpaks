# wh3lk submodular power pack

A lil eurorack power module, primilary for prototyping. Takes 16 pin eurorack power and outputs:

- +12V
- +5V
- +3.3V
- +3.3VA (filtered for analog devices)
- -12V
- -10V reference
- GND
